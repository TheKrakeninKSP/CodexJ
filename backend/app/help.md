# CodexJ Help

CodexJ is organized around workspaces, journals, and entries.

## Navigation overview

- Sidebar: Main navigation for workspaces, journals, account actions, imports/exports, and app help.
- Workspace view: Your top-level area for selecting where journals live.
- Journal view: Shows entries for the selected journal.
- Entry editor: Create or update entry content.
- Entry reader: Read a full entry with its metadata.

## Sidebar button guide

- Create Workspace: Adds a new workspace.
- Create Journal: Adds a new journal inside the selected workspace.
- Help: Opens this help page.
- Log Out: Ends your session and sends you to login.
- Export: Exports your encrypted data dump (Privileged mode required).
- Import Entry: Imports one plaintext `.txt` entry and optional media files into the selected journal.
- Trim Media: Deletes unreferenced media files (Privileged mode required).
- Shred: Exports your data and permanently deletes your account (Privileged mode required).

## Privileged mode

Privileged mode unlocks sensitive actions:

- Deleting workspaces and journals.
- Exporting encrypted data.
- Trimming orphaned media.
- Running Shred.

Use the Privileged mode button in the sidebar, re-enter your password, and confirm.

## Basic usage flow

1. Create a workspace.
2. Create a journal in that workspace.
3. Open the journal and create an entry.
4. Add media (images, videos, audios) to your entry.
5. Save entry (entry name is optional but entry type is required).

## Tips

- Select a journal before using Import entry.
- Keep your encryption key safe during export.
- Use Trim media occasionally to reclaim disk space.
